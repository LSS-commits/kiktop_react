# Kiktop App

TikTok App clone (desktop) in React with database using: 
- [Stargate REST API](https://stargate.io/) 
- [DataStax Astra database](https://www.datastax.com/)
- [Netlify (Jamstack)](https://www.netlify.com/jamstack/)
